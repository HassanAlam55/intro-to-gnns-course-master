## Overview 

The Theory section of the course covers the foundations of graph neural networks.  We discuss the message passing framework and how this addresses the many challenges that arise from symmetries in graphs.  We work our way up to understanding convolutions on graphs, and where they come from.

### Finding things

If you just want the videos, you can see them on [this YouTube playlist](https://www.youtube.com/playlist?list=PLSgGvve8UweEEhiJO_jTcyWWRTFVOFk9Z).  

Within this repo, each lecture is encapsulated by a directory in which the README has a link to the video and slides along with a quiz and some additional references.  A separate markdown file contains the answers to the quiz questions.
