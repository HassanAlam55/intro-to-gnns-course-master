## Overview 

The Hands-on portion of the course covers the three most popular GNN application areas:

1. Node Classification
2. Link Prediction
3. Graph Classification

Each section contains:

* An overview video that introduces the problem framing
* An exercise in the form of a notebook with "TODO" comments that indicate the sections that need filled in by the student
* An answer key notebook where the TODO sections are filled in with correct answers
* A solution walk-thru video that explains the answer key
